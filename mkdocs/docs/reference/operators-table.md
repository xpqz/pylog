# Operators table (overview)

Common operators and categories:

- Term: `=`, `\=`
- Arithmetic compare: `=:=`, `=\=`, `<`, `=<`, `>`, `>=`
- Arithmetic eval: `is`
- Arithmetic: `+`, `-`, `*`, `//`, `mod`
- Control: `,`, `;`, `->`

For precedence and associativity, see the implementation notes and tests under operators.
