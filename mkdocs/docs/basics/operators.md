# Operators

PyLog supports standard Prolog operators with defined precedence and associativity.

- Arithmetic: `+`, `-`, `*`, `//`, `mod`
- Comparison: `=:=`, `=\\=`, `<`, `=<`, `>`, `>=`
- Term: `=`, `\\=`, `=..`
- Control: `,` (conjunction), `;` (disjunction), `->/2` (if‑then)

See Reference → Operators table for details.
